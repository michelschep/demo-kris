## Why

The project needs a visual landing page that displays the name "Kris" in a creative, animated p5.js sketch using red and blue colors. This establishes the visual identity of the site.

## What Changes

- Add `index.html` as the main entry point
- Add `sketch.js` containing a p5.js sketch that renders "Kris" in red and blue
- Add `style.css` for basic full-screen canvas layout

## Capabilities

### New Capabilities
- `kris-display`: Renders the name "Kris" on a p5.js canvas using red and blue colors, centered on the page
